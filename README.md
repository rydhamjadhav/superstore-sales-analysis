# superstore-sales-analysis
Superstore Sales Analysis | SQL - Excel - Power BI - BigQuery

- Analyzed 9,994 orders across 4 years using BigQuery SQL,
  writing 6 queries covering KPIs, trends, and discount impact.

- Cleaned and enriched raw data in Excel adding calculated
  columns for Profit Margin, Delivery Days, and Discount Band.

- Built a 3-page interactive Power BI report covering Sales
  Overview, Category Analysis, and Regional Performance.

- Discovered that orders with discounts above 20% consistently
  generated negative profit.
